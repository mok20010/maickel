About me

Une méthode de la nouvelle génération

Notre auto-école est agréée par l'état, vous pouvez donc vous former en ligne et passer le code de la route et l'examen du permis de conduire. Grâce à cette méthode nouvelle génération, nous réduisons considérablement le prix du permis de conduire. Lors d'un changement d'auto-école, sachez que nous ne facturons aucun frais de transfert de dossier et pouvons vous renseigner sur la procédure de restauration de votre dossier d'auto-école. Inscrivez-vous en ligne et choisissez la formule qui vous convient pour débuter votre formation : code en ligne, cours de conduite ou permis complet... Vous pouvez personnaliser votre formation selon vos besoins. Avec notre auto-école en ligne, vous pouvez vérifier le code valable pour toutes les classes de permis de conduire.
